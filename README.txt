== Description ==

BulletProof Payment Gateway is a WooCommerce payment gateway integration that enables users to make payments through the BulletProof API.

**Features:**

- Seamless integration with WooCommerce.
- Secure payments using the BulletProof API.
- Sale authorization and order processing.

== Installation ==

1. Upload the `bulletproof-payment-gateway` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to WooCommerce > Settings > Payments and configure your BulletProof Payment Gateway settings.

== Configuration ==

1. Visit the WooCommerce settings page and click on the "Payments" tab.
2. Find "BulletProof Payment Gateway" in the list of available payment methods and click "Manage."
3. Configure the necessary settings, such as API credentials, security key, etc.
4. Save changes.

== Usage ==

1. During the checkout process, customers will see "BulletProof Payment Gateway" as a payment option.
2. Customers provide necessary payment details and complete the order.
3. The plugin processes the payment through the BulletProof API.
4. Order status is updated based on the API response.

== Frequently Asked Questions ==

= Is this plugin secure? =

Yes, the plugin utilizes the BulletProof API for secure payment processing.

= Where can I get API credentials? =

You need to sign up for an account with BulletProof and obtain API credentials (username, password, security key).
